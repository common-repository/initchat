initChat will help to start quick support on your website. this plugin will work with third party application of WhatsApp.   

== Description ==
Integrate Free WhatsApp chat support dialog directly into your WordPress website and that will help you to increase leads/sales from your website. this is well design with lightwight code help to attract users for chat with active agent using WhatApp application. 
Now a days most of users are using WhatsApp application and this plugin will initiate chat.

= Shortcodes for inline initChat widget =

> `[initChat]` 
> `[initChat_Dialog]` 


= Available fields =
- Third party application (WhatsApp) mobile Number.
- Agent Image
- Agent Name
- Agent Status
- Welcome Message
- Appearance 
- Setting
- Dialog Options
- Shortcodes

- Content (In content you can change your messages)

== Installation ==

= Installation =
1. In your WordPress admin panel, go to Plugins > New Plugin
2. Search Initchat(WhatApp) Chat plugin by Initchat and click Install now
3. Alternatively, download the plugin and upload the contents of initchat.zip to your plugins directory, which usually is /wp-content/plugins/
4. Activate the plugin
5. Enjoy
